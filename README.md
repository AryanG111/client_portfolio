# Digital Marketing Portfolio

This is a beautiful, animated React portfolio designed for Swapnil Haramkar. It features scroll-triggered animations and a premium dark-mode aesthetic with vibrant accent colors.

## Setup Instructions

Currently, Node.js and NPM are required to run this project. Since they were not found in this environment, please install them first if you haven't already.

1. **Install Node.js & NPM:** Download from [nodejs.org](https://nodejs.org/).
2. **Install Dependencies:** Open your terminal in this folder and run:
   ```bash
   npm install
   ```
3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
4. Open your browser to the URL provided (usually `http://localhost:5173`).

## Technologies Used
- **React (Vite)**
- **Framer Motion** (for beautiful scroll animations)
- **Vanilla CSS** (for precise, flexible styling)
- **Google Fonts** (Inter, Playfair Display, Great Vibes)
